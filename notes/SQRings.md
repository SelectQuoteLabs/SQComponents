# SQRings

## Usage

```jsx
import {SQRings} from 'scplus-shared-components';
```

```jsx
const someComponent = () => {
  <>
    <SQRings height="100" />
    {/* more awesome components*/}
  </>;
};
```

## Technical Details

### Props

| Name   | Type     | Description            | Required | Default |
| ------ | -------- | ---------------------- | -------- | ------- |
| height | `string` | The height of the logo | No       | `'283'` |
